# Freebox Ultra – NetworkControl Integration

![logo](logo.png)

Cette intégration permet :

- Découverte automatique des profils Freebox Ultra (OS 15)
- Création d’un switch par profil
- Pause / reprise Internet
- Icônes Freebox natives affichées dans Home Assistant
- Affichage des appareils connectés à chaque profil
- Authentification automatique par HMAC v8
- Support complet Freebox Ultra via NetworkControl

Développé pour Freebox Ultra (Freebox OS 15).
